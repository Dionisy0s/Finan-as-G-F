# Finanças G&F

Este é um projeto de controle financeiro avançado que substitui uma planilha financeira, oferecendo análises detalhadas, dashboards, gráficos interativos e armazenamento offline. Ele permite o gerenciamento financeiro para duas pessoas com funcionalidades de sincronização, relatórios e integração de IA para análise de dados.

## Funcionalidades
- Controle e categorização de entradas e saídas financeiras.
- Dashboards interativos com gráficos em tempo real.
- Opções para modo claro e escuro.
- Funciona online e offline com PWA (Progressive Web App).
- Relatórios customizáveis em diferentes formatos.
- Gestão de categorias e metas de investimento.
- Análises recorrentes assistidas por IA para otimização financeira.

Este repositório contém o código para o desenvolvimento do site. Mais detalhes serão adicionados conforme o progresso do projeto.
